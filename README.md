# Star Encounter - 星之邂逅

一款二次元角色互动与养成APP，让玩家与虚拟角色建立深度连接。

## 项目简介

Star Encounter 是一款以二次元文化为核心的角色互动应用，玩家可以：
- 与多个原创二次元角色进行对话互动
- 培养角色的好感度和能力
- 探索丰富的故事剧情
- 收集角色羁绊和成就

## 核心特性

- 🎭 **多角色系统** - 每个角色都有独特的性格、背景和对话风格
- 💬 **深度对话** - 基于NLP的对话系统，支持上下文理解和情感分析
- 🎮 **养成系统** - 角色升级、技能解锁、装备收集
- 📖 **剧情驱动** - 分章节的沉浸式故事体验
- 🎨 **精美UI** - 二次元风格界面，流畅动画效果

## 技术栈

- **前端**: Flutter 3.x
- **后端**: Node.js + Express
- **数据库**: SQLite (本地存储)
- **AI对话**: 自定义对话引擎 + 情感分析
- **版本控制**: Git + GitHub
